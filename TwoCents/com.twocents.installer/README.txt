INFORMAÇÕES 

1. Para poder gerar o installer é necessário alguns passos:

- Instalar o NSIS(http://nsis.sourceforge.net/Download) - Estamos trabalhando com a versão 2.46
- Após a instalação, é necessário colocar no PATH do Windows o diretorio do NSIS, pq o ANT vai vai ser uma chamada local do executavel.

2. Para desenvolvimento:

- Para auxiliar na construção dos scrips NSIS, existe um plugin do eclipse(http://eclipsensis.sourceforge.net/update)
